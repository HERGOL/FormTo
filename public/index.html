Documentation API

Intégrez l'envoi d'e-mails dans n'importe quelle app.

Endpoint
POST /api/send

La clé API doit être transmise dans le corps JSON de la requête, sous le champ api_key. Le header Content-Type: application/json est obligatoire.

Paramètres du corps (JSON)
Champ Type Obligatoire Description
api_key string Oui Votre clé API sk_live_…
to string Oui Adresse e-mail du destinataire
subject string Oui Sujet du message
text string Non Corps en texte brut
html string Non Corps en HTML
from string Non Surcharge l'expéditeur (domaine autorisé requis)
cURL
curl -X POST http://localhost:3000/api/send \ -H 'Content-Type: application/json' \ -d '{ "api_key": "sk_live_VOTRE_CLE", "to": "client@example.com", "subject": "Hello", "text": "Contenu" }'
Python
import requests r = requests.post( 'http://localhost:3000/api/send', json={ 'api_key': 'sk_live_VOTRE_CLE', 'to': 'client@example.com', 'subject': 'Hello', 'text': 'Contenu' } )
JavaScript (fetch)
fetch('http://localhost:3000/api/send', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ api_key: 'sk_live_VOTRE_CLE', to: 'client@example.com', subject: 'Hello', text: 'Contenu', html: '<h1>Contenu</h1>' }) })
PHP
<?php $ch = curl_init('http://localhost:3000/api/send'); curl_setopt($ch, CURLOPT_RETURNTRANSFER, true); curl_setopt($ch, CURLOPT_POST, true); curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode([ 'api_key' => 'sk_live_VOTRE_CLE', 'to' => 'client@example.com', 'subject' => 'Hello', 'text' => 'Contenu', ])); curl_setopt($ch, CURLOPT_HTTPHEADER, [ 'Content-Type: application/json' ]); echo curl_exec($ch);
Réponses
200 Succès
{ "success": true, "message_id": "<uuid@host>" }
403 Clé invalide
{ "error": "Clé API invalide" }
